## What is akenodev.xyz ?

[akenodev.xyz](https://akenodev.xyz) is the website for [**AKENO**](https://github.com/akenolol) the owner of **RED CAFE**


## How to setup ?

Clone the project

```bash
  git clone https://github.com/akenolol/akenodev.xyz
```

Go to the project directory

```bash
  cd akenodev.xyz
```

Install dependencies

```bash
  npm install
```

Edit `config.js`

```bash
vim config.js
```

```bash
code config.js
```

Start the server

```bash
  npm run start
```
