# Goddess Bot


## What is Goddess Bot ?

Goddess Bot is the **OFFICIAL** Discord Bot For Red Cafe !


## How to setup ?

Clone the project

```bash
  git clone https://github.com/Red-Cafe/Goddess-Bot.git Goddess-Bot
```


Go to the project directory

```bash
  cd Goddess-Bot
```

Install dependencies

```bash
  npm install
```

pen config.json, and change everything

```json
{
    "port": "8080",
    "ownerID": "your-id",
    "guildID": "your-guild-id",
    "clientID": "your-client-id",
}
```


Open sample.env, and add your things.


## Building

```bash
node deploy.js && npm run dev
```

## What You Will Need.

[MongoDB](https://account.mongodb.com/account/login)

[Discord Developers](https://discord.com/developers/applications)
